■ ステータスの意味

┌────────────┬─────────────────────────────────────┐
│ ステータス │ 意味                                │
├────────────┼─────────────────────────────────────┤
│ 未使用     │ 準備済み、顧客未割当                │
│ 送信待ち   │ → これに変更すると招待メールが送信  │
│ 招待済     │ メール送信完了、LINE連携待ち        │
│ 契約済     │ LINE連携完了、領収書受付中          │
└────────────┴─────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 新規顧客を登録する手順

【STEP 1】顧客管理シートを開く

【STEP 2】「未使用」ステータスの行を探す

【STEP 3】以下を入力
  ・B列: 顧客名
  ・I列: メールアドレス

【STEP 4】H列（ステータス）を「未使用」→「送信待ち」に変更
  ※ ドロップダウンから選択

【STEP 5】自動で招待メールが送信される
  ・成功 → ステータスが「招待済」に自動変更
  ・失敗 → エラーダイアログが表示され、ステータスは「未使用」に戻る

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 顧客がLINE連携する流れ（顧客の作業）

1. 顧客が招待メールを受信
2. LINEで公式アカウントを友達追加
3. トーク画面で顧客コード（例: KZ001）を入力
4. 「設定が完了しました」と表示される
5. ステータスが「招待済」→「契約済」に自動変更

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 領収書・通帳の送信（顧客の作業）

1. 顧客がLINEで写真を送信
2. 自動で分類されて保存
   ・レシート → 領収書フォルダ
   ・通帳 → 通帳フォルダ
3. 顧客に確認メッセージが返信される

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
